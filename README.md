# Vitto × IDBI – Prospect Assist AI

AI-powered relationship manager assistant that helps banks identify, qualify, and engage high-potential prospects with intelligent insights and personalized recommendations.

## 🚀 Live Demo

https://vitto-idbi.lovable.app

## ✨ Features

- AI-powered prospect profiling
- Intelligent lead qualification
- Customer insights dashboard
- Personalized product recommendations
- Relationship Manager copilot
- Conversational AI interface
- Fast, intuitive, and responsive UI

## 🎯 Problem

Relationship Managers spend significant time analyzing customer information, identifying opportunities, and recommending suitable banking products. This process is often manual, fragmented, and time-consuming.

## 💡 Solution

Prospect Assist AI streamlines customer discovery by combining AI-driven insights with a conversational interface, enabling RMs to make faster, smarter, and more personalized engagement decisions.

## 🛠 Tech Stack

- React
- TypeScript
- Tailwind CSS
- Lovable

## 📌 Status

Prototype built for the **IDBI Bank Hackathon**.

> **Note:** This is a proof-of-concept prototype created for demonstration purposes and is not connected to live banking systems.
